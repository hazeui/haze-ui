# Button 
Clickable Element

## Variants
Add the btn-variant class to set the variant, example: btn-soft  
<ToggleDemo demo={getDemo("btn", "variant")} />

## Icons    
Use the iconL and iconR prop, copy icon classes from <u>[iconify](https://icon-sets.iconify.design/)</u>  
<ToggleDemo demo={getDemo("btn", "icons")} />

## Sizes and class  
Add the btn-variant-size class to change the size of the button. example: btn-primary-sm  
<ToggleDemo demo={getDemo("btn", "size")} />

## Classes  
Add the btn-variant class to set the variant, example: btn-soft  
<ToggleDemo demo={getDemo("btn", "class")} />
