# Input 

Flexible and styled input component with icon support, size variants, and focus management

## Usage

Use `grinput-<variant>-<size>` to style the group input

<ToggleDemo  demo={getDemo("input", "usage")} />

## Sizes

The 2nd class is optional

<ToggleDemo class='grid demobox-20' demo={getDemo("input", "size")} />

## Form

The 2nd class is optional

<ToggleDemo class='demobox-20' demo={getDemo("input", "form")} />

## Html only

If you dont want to add icons in your input then just use "input" class

<ToggleDemo class='flex-col' demo={getDemoHtml("input", "html")} />
